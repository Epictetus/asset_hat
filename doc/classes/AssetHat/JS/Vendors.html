Module AssetHat::JS::Vendors

Adds support for automatically popular loading third-party JavaScript either locally or from a remote CDN based on the environment.

Public class methods

source_for (vendor, options={})

Arguments:

vendor
A symbol in VENDORS.
options
The only supported option is :version. This overrides the vendor version given in the config file, which you may want to do in large apps with multiple layouts and multiple dependency versions.
